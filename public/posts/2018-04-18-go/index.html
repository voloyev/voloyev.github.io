Pointers

In go everything pass by value. That means when you pass value as argument to function go copies the value.

Example:

type person struct{
    name string
    age int
}

bob := person{name: "Bob", age: 22,}


func(p person) changeName(newName string){
    p.name = newName
}

bob.changeName("Alice")

// we expect that function will change name from Bob to Alice
// But no

fmt.Println(bob.name) 
// return bob

to fix this we need to change argument

    func (p *person) changeName(newName string) {
        p.name = newName
    }
    
    //and than here two options:
    //1: more explicit
    pointer := &bob
    pointer.changeName("Alice")
    //2: implicit
    bob.changeName("Alice")
